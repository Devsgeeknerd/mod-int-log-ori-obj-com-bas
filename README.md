# Tabelas

```html
  <table></table>
```

- Atributos:
  - ***width***:
    - É a largura da tabela.
  - ***border***:
    - É a borda da tabela.
  - ***colspan***:
    - Meclar as colunas.
  - ***rowspan***:
    - Mesclar as linhas.

```html
  <tr></tr>
```

  > Inseri as linhas.

```html
  <td></td>
```

  > Inseri as colunas.

- ***align***:
  - Alinha o texto que pode ser:
    - ***left***: Esquerda.
    - ***right***: Direita.
    - ***center***: Centro.

- ***bgcolor***:
  - Pode ser usado tanto na linha como na coluna.
